## What does Copilot do?

**Analyzes the context in the file you are editing, as well as related files, and offers suggestions from within your text editor.**

- autocomplete code
- convert comments to code
- powered by Codex (AI system created by OpenAI)
- trained on 1.5 billion lines of code

Notes: Cloud based AI tool developed by GitHub and OpenAI

Released as a beta in June 2021

Optimized to help you write Python, JavaScript, TypeScript, Ruby, Go, C#, or C++

Extension for VS Code, Visual Studio, and JetBrains IDEs
